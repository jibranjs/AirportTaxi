# Vuejs3 Skeleton v1.0

A modern Vue.js 3 starter template with PrimeVue 4 integration.

## Technologies Used

- [Vue.js 3](https://vuejs.org/) - Progressive JavaScript Framework
- [PrimeVue 4](https://primevue.org/) - UI Component Library
- [PrimeIcons](https://primevue.org/icons) - Icon Library
- [Iconify](https://iconify.design/) - Icon Framework
- [Pinia](https://pinia.vuejs.org/) - State Management
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS Framework
- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- JavaScript

## Project Structure

```
src/
├── App.vue                 # Root component
├── main.js                 # Application entry point
├── assets/                 # Static assets
├── components/
│   └── layout/
│       ├── AppHeader.vue   # Application header component
│       └── AppFooter.vue   # Application footer component
├── router/
│   └── index.js           # Vue Router configuration
├── stores/                 # Pinia stores
└── views/
    └── HomeView.vue       # Home page view
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
